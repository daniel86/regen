Detailed Description

A sprite that generates sphere normals.

It is not possible to define per vertex attributes for the sphere because each sphere is extruded from a single point in space. You can only add per sphere attributes to this mesh. This is a nice way to handle spheres because you get a perfectly round shape on the render target you use and you can anti-alias edges in the fragment shader easily.
