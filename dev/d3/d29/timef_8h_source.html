43// TimeF manages an osg::Timer and features an interface for floating time
44// in the closed interval [0;1] representing a full day and standard c
45// time (time_t) simultaneously. The time updates have to be requested
46// explicitly, thus simplifying usage between multiple recipients.
47// The time starts cycling automatically, but can also be paused, stopped,
48// or set to a specific value.
