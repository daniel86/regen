const string & Includer::include ( const std::string &  key)

Include section from shader file. The include key contains sub directory names relative to include path followed by the shader file name and a sequence identifying a section within the file. All items are separated by dots. For example "regen.picking.vs" loads section 'vs' from file 'picking.gls' that is in a child-directory, of an include path, named 'regen'.

Parameters
keythe include key.
Returns
the shader section or empty string.
