Detailed Description

Atmospheric scattering, and some simple cloud rendering. A cube map is created on updated, and then rendered each frame using a sky cube.

See also
http://codeflow.org/entries/2011/apr/13/advanced-webgl-part-2-sky-rendering/
http://http.developer.nvidia.com/GPUGems2/gpugems2_chapter16.html
https://github.com/shff/opengl_sky
